Welcome to Boundless Skies Node Agent

This installer will set up the Boundless Skies Node Agent on your Mac.

The Node Agent runs in the background as a system service, automatically:

  • Connecting to your Seestar telescope
  • Downloading tonight's observation plan
  • Processing photometry and uploading science data

Requirements:

  • macOS 11 or later
  • Seestar S50 or S30 in Station Mode on your local WiFi
  • A Boundless Skies activation code (get one at boundlessskies.org)

Everything else — Python, the plate solver, all science libraries — is included. No separate downloads required.

After installation, the dashboard opens automatically at http://localhost:5173. A one-time setup wizard will guide you through connecting your Seestar and downloading the star catalog for accurate plate solving (~6 GB, downloaded in the background).
